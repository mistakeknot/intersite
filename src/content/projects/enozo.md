Enozo is inspired by a remarkable piece of software in the late 1990s: iZotope Ozone for Winamp. With the press of a button, music sounded markedly different, but not in a way that could be easily described. The plugin ecosystem for something like Winamp died just as things like Winamp did, but the surreal, synthetic vinyl warmth from the Ozone plugin can now be brought back to the desktop (more likely laptop) via Enozo's hooks into macOS Core Audio. Running a five-stage mastering chain across all system audio output to create a subtle effect only noticed by its absence.
